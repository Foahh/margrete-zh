# 改变音符的属性

使用属性编辑功能，可以改变 ExTAP 的效果或有无 AIR-HOLD 的终点等。

## 更改方法

1. 选择编辑工具栏的 ![属性編集](/img/docs/edit-toolbar-attr.png) ，切换到属性编辑模式。

2. 在谱面编辑区域内，左键点击要改变属性的音符。

3. 属性设置菜单会显示出来，可以勾选或取消勾选相应的选项。

![edit note attribute](/img/docs/edit-note-attr.png)

## 可以改变的音符和内容

| 音符                 | 可以改变的内容            |
| -------------------- | ------------------------- |
| ExTAP                | ExTap 的击打背景特效      |
| FLICK                | AutoPlay 时的击打方向特效 |
| AIR                  | AIR 方向、颜色反转        |
| AIR-HOLD 终点        | 音符的有无                |
| AIR-SLIDE 终点       | 音符的有无                |
| AIR-CRUSH 起点、终点 | 音符的有无                |
| AIR-CRUSH 起点       | 装饰线的颜色              |

> Tips: 可以在插入模式 ![笔](/img/docs/edit-toolbar-pen.png) 下**双击**音符来删除 AIR-HOLD 终点、AIR-SLIDE 终点、AIR-CRUSH
