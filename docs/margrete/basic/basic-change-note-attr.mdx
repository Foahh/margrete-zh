# 修改音符属性

## 修改步骤

1. **切换到属性编辑模式**
- 点击编辑工具栏中的 <Icon src="/img/docs/edit-toolbar-attr.png" alt="属性编辑" /> 图标。

2. **选择要修改的音符**
- 在谱面编辑区域内，左键点击目标音符。

3. **设置属性**
- 此时将会弹出属性设置菜单，请根据需要勾选或取消相关选项。

    <Image src="/img/docs/edit-note-attr.png" alt="编辑音符属性"/>

## 可修改的音符类型及其属性

| 音符类型               | 可修改内容                  |
| ---------------------- | --------------------------- |
| **ExTap**              | ExTap 的击打背景特效        |
| **Flick**              | AutoPlay 模式下的击打方向特效 |
| **Air**                | Air 的方向及颜色反转        |
| **Air-Hold 终点**      | 是否存在该终点              |
| **Air-Slide 终点**     | 是否存在该终点              |
| **Air-Crush 起点/终点** | 是否存在起点及终点          |
| **Air-Crush 起点**     | 装饰线的颜色                |

:::tip
在**插入模式** <Icon src="/img/docs/edit-toolbar-pen.png" alt="笔" /> 下，直接**双击**音符就可以删除 Air-Hold 终点、Air-Slide 终点以及 Air-Crush 的起点和终点。
:::
