# 编辑谱面属性

## 说明

按照以下步骤编辑谱面属性：

1. 在菜单栏中点击 **File**

2. 选择 <Icon src="/img/docs/file-chart-prop.png" alt="Chart Property" /> **Chart Property**。

3. 在弹出的窗口中填写所需内容，例如曲师、曲名、难度等。

4. 点击 **OK** 以保存并关闭窗口。

> **注意**：直接关闭窗口将不会保存已填写的内容，请务必点击 **OK**。

> **重要提示**：请勿留空 **Song ID**，通常由 Margrete 自动填写此项。

部分项目说明:

![chart property](/img/docs/chart-prop.png)

> 注意：修改 **Basic BPM** 并不会改变谱面 BPM，因此请注意要在开头[插入 BPM 更变事件](/docs/margrete/basic/basic-edit-chart)

## 设置音源

1. 确保音源文件和谱面文件都在一个文件夹内

2. 将音源名（带扩展名）填写到**BGM filename**里。

![音源位置](/img/docs/audio-example.png)

> Tips: 推荐使用 wav 格式作为音源

1. 在设置好音源和 BPM 后，你可以通过试听来检查音符的时间是否和音源一致。

- 点击标准工具栏的 ![播放](/img/docs/std-toolbar-play.png) 或者按键盘的<kbd>Space</kbd> 键，就可以从光标（红线）的位置开始播放。再次进行同样的操作，就可以停止播放。

- 当音源对不上的时候，可以填入音源延迟来调整（单位：秒）。
