# 编辑谱面事件

:::tip[快捷键]
    熟练使用快捷键可以大幅提高写谱效率，通常可以在菜单选项的右侧看到快捷键提示。
:::

## 设置不同的时间线 (TIL)

为了实现特殊的变速效果，可以设置不同的时间线 **(TIL)**。

- 在 **编辑工具栏** 的右边找到与 **TIL** 相关的下拉菜单。

  <Image src="/img/docs/change-current-til.png" alt="设置下落速度组" />

- 设置的下落速度将与选定的时间线相关联。默认时间线为 **TIL 0**。


## 插入 BPM 变更事件

1. 选择编辑工具栏的 ![选择](/img/docs/edit-toolbar-select.png) ，切换到选择模式。

2. 点击要插入的位置来移动光标 (红线)。

3. 点击菜单的 **Insert** > **BPM Change Event**。

4. 在弹出的窗口中输入 BPM 值，点击确定。

> Tips: 按下 <kbd>B</kbd> 可以快速插入 BPM 变更事件

## 插入流速变更事件

1. 选择编辑工具栏的 ![选择](/img/docs/edit-toolbar-select.png) ，切换到选择模式。

2. 点击要插入的位置来移动光标 (红线)。

3. 点击菜单的 **Insert** > **Scroll Speed Change Event**。

4. 在弹出的窗口中输入变速倍数，点击确定。

> Tips: 按下 <kbd>N</kbd> 可以快速插入流速变更事件

## 插入拍子变更事件

1. 选择编辑工具栏的 ![选择](/img/docs/edit-toolbar-select.png) ，切换到选择模式。

2. 点击要插入的位置来移动光标 (红线)。

3. 点击菜单的 **Insert** > **Time Signature Event**。

4. 在弹出的窗口中输入拍号，点击确定。

> Tips: 按下 <kbd>M</kbd> 可以快速插入拍子变更事件

## 批量删除事件

1. 同时按键盘上的 <kbd>Shift</kbd> 和 <kbd>Delete</kbd> 键，或者点击菜单中的 **编辑** 👉 **删除选中事件** 。

2. 选择范围内的事件会被删除。
