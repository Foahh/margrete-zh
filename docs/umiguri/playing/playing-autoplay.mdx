# 游玩或预览 UMIGURI 谱面

- 此界面下 UMIGURI 已配置为中文

## Auto Play

- 选择曲目后 👉 **Details** 👉 **Game** 👉 **Auto Play**

- ![Select](/img/docs/umiguri-select.png)

- ![Auto](/img/docs/umiguri-auto.png)

## 写谱模式

- 在选曲界面按下<kbd>F4</kbd>进入写谱模式，再次按下<kbd>F4</kbd>则退出写谱模式。

  ![editor1](/img/docs/umiguri-editor.png)

- 写谱模式下可调整播放时间，更改流速/播放速度，以及检查无理配置。

  ![editor2](/img/docs/umiguri-editor2.png)

- 使用方法

  |              按键               | 作用             |
  | :-----------------------------: | ---------------- |
  |          <kbd>←</kbd>           | 后退一小节       |
  |          <kbd>→</kbd>           | 前进一小节       |
  | <kbd>Shift</kbd> + <kbd>←</kbd> | 后退 1 秒        |
  | <kbd>Shift</kbd> + <kbd>→</kbd> | 前进 1 秒        |
  |          <kbd>↑</kbd>           | 增加谱面流速     |
  |          <kbd>↓</kbd>           | 降低谱面流速     |
  | <kbd>Shift</kbd> + <kbd>↑</kbd> | 加快播放速度     |
  | <kbd>Shift</kbd> + <kbd>↓</kbd> | 减慢播放速度     |
  |        <kbd>Space</kbd>         | 播放 / 停止      |
  |          <kbd>F3</kbd>          | 切换自动 / 手动  |
  |          <kbd>F5</kbd>          | 重新加载谱面     |
  |          <kbd>F6</kbd>          | 从头开始         |
  |          <kbd>F9</kbd>          | 回到选曲画面     |
  |         <kbd>F10</kbd>          | 判定区域显示切换 |

> Tip: 可以利用 <kbd>F5</kbd> 在 UMIGURI 和 Margrete 之间快速预览谱面更改
